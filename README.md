# Customer-Behaviour-Dashboard-Power-BI
This project presents an interactive Customer Behaviour Dashboard built using Power BI to analyze customer purchasing patterns and business performance
Key Insights Covered

Average customer rating, total customers, and average purchase amount
Subscriber vs non-subscriber analysis
Revenue and customer count by product category
Purchase behavior across different age groups
Review ratings segmented by age group
Filters for subscription status, category, and shipping type
--Tools & Technologies
Power BI
Data modeling & DAX measures
Interactive visuals and slicers
--objective
To transform raw customer data into meaningful insights that help understand customer behavior, improve decision-making, and support business strategy.
This repository contains the Power BI (.pbix) file and supporting resources.
